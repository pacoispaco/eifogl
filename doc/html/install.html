Installing EiffelOpenGL

Introduction

EiffelOpenGL was originally developed with ISE EiffelBench 4.2f on Windows 95/NT and that it does to some extent rely on WEL. All of the EiffelOpenGL example applications have compiled and run without any problems under ISE EiffelBench 4.5.
 

Installing the EiffelOpenGL archive file

Unzip the eifogl0_1_0.zip file into a directory of your choice. This directory will henceforth be referred to as the EiffelOpenGL directory.
 

Environement variables

You must manually set the following environment variables:

EIFOGL                            The EiffelOpenGL directory.
EIFFEL_COMPILER        Your Eiffel compiler (One of: ise, smalleiffel).
GL                                     Directory where the "include" and "lib" directories (containing the import libraries) for OpenGL and AUX reside.
GLUT                                Directory where the "include" and "lib" directories (containing the import library) for GLUT reside.
